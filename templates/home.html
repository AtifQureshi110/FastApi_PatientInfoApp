Mini Patient Management System

This is a mini project built using FastAPI â€” a modern web framework for building APIs with Python. The project follows the principles of a RESTful API, allowing standardized interaction between the client and server.

Features:

  • Add new patient records
  • Update existing patient information
  • Delete patient records
  • View all stored patient entries

All data is stored in a Microsoft SQL Server Management Studio (SSMS) database.

FastAPI & REST:

FastAPI is used to create RESTful API endpoints that handle HTTP requests (GET, POST, PUT, DELETE). Each operation is mapped to a specific route using REST principles.
